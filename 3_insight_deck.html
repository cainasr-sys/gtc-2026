07 — The Rise of AI Agents
The Rise of AI Agents:
Navigating AI Agent Architecture & Governance

From Models to Agents

  • Falling inference costs and advances in reasoning are fueling the rise of agentic systems
  • Agents mark a shift from static copilots to autonomous, multi-step executors
  • Emerging standards like MCP make scalable, deterministic deployment possible

AI Agents Are Evolving Enterprise Architecture

  • Agents fuse applications and automation, embedding reasoning directly into workflows
  • New agent stacks are emerging with memory, identity, observability, and orchestration layers
  • Incumbents retrofit with copilots, while startups reinvent processes through agentic design

The Security & Governance Imperative

  • Agent adoption introduces new risks in identity, authorization, and data exposure
  • AI-driven threats demand innovation beyond legacy security models
  • Opportunities span agent IAM, observability, integration security, and data governance

DEFINING AN AI AGENT
  • An Agent is an application which executes a task or function autonomously using tools
  • An Agent should be able to take NL input and reason on it to break it down into a Task plan
  • Agents are centered around an LLM with the ability to use tools & APIs to access data or execute code
  • Code they execute can be static application code, predictive ML models or a LLM call
OF AN AGENTIC ARCHITECTURE — KEY COMPONENTS
  • Deterministic run time
  • Reflection loops enable Agent to evaluate, output, self correct using guardrails
  • Human in the Loop

09 — Considerations for Implementors
Some considerations for implementors

AI copilots are a great way for incumbent vendors to incorporate GenAI

  • Improved UX & bootstrapping for users
  • LLMs simplify tasks within the application
  • Focus on improving user productivity

Applications & automation converge

  • Deep understanding of a workflow key
  • Agents enable rethink of Application architectures by subsuming workflows
  • Human in the Loop

We're still in the early days of AI driven Architectures

  • Model capabilities & AI tooling evolving rapidly
  • Economic equation redefined w/ GenAI apps
  • Cool demos ≠ Working products
